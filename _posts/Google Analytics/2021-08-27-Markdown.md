### 3. Italic Text
```markdown
_text_
*text*
```
**Return**
<html>
<head>
<style type="text/css">
.text-red {color: firebrick; font-style: italic;}
</style>
</head>
<body>
<div class="text-red">
    text
</div>
</body>
</html>
<br/>

### 4. Bold Text
```markdown
__Text__
**Text**
```
**Return**
<html>
<head>
<style type="text/css">
.text-red-bold {color: firebrick; font-weight: bold;}
</style>
</head>
<body>
<div class="text-red-bold">
    bold
</div>
</body>
</html>
